# Products Dashboard

SPA-приложение для управления товарами. Реализована авторизация, просмотр списка товаров, поиск, сортировка, добавление новых товаров и пагинация.

Проект выполнен на React + TypeScript с использованием современного фронтенд-стека.

---

## Стек технологий

- React
- TypeScript
- Vite
- TailwindCSS
- TanStack React Query
- React Hook Form
- Zod
- Zustand
- Lucide React (иконки)
- Sonner (toast-уведомления)

---

## Функциональность

### Авторизация

Реализована форма входа пользователя.

Возможности:

- валидация полей
- обработка ошибок API
- скрытие / показ пароля
- запоминание сессии

Логика хранения токена:

| Чекбокс | Хранилище |
|-------|----------|
| включен | localStorage |
| выключен | sessionStorage |

После успешной авторизации происходит переход на страницу товаров.

Тестовые данные для входа:

username: emilys  
password: emilyspass

---

### Страница товаров

Основная страница отображает таблицу товаров, полученных из API.

Возможности:

- загрузка товаров из API
- поиск товаров
- сортировка
- пагинация
- добавление товара
- выделение товаров чекбоксами
- обновление данных

---

### Поиск товаров

Поиск реализован через API.

Особенности:

- используется debounce 400ms
- запрос отправляется только если введено 3 и более символов

---

### Сортировка

Поддерживается сортировка по столбцам:

- наименование
- вендор
- артикул
- рейтинг
- цена

Состояние сортировки хранится в React state.

---

### Добавление товара

Кнопка «Добавить» открывает форму.

Можно добавить:

- название
- цену
- вендора
- артикул

После успешного добавления:

- товар появляется в таблице
- отображается toast-уведомление

Сохранение на сервер не требуется по условиям задания.

---

### Подсветка рейтинга

Если рейтинг товара ниже 3, значение автоматически подсвечивается красным цветом.

---

### Пагинация

Реализована динамическая пагинация.

Особенности:

- отображается 20 товаров на странице
- текущая страница отображается по центру

Пример:

1 2 3 4 5  
3 4 5 6 7  
5 6 7 8 9

---

## API

Используется публичный API:

https://dummyjson.com/docs/products

Основные endpoints:

Получение товаров

GET /products

Поиск товаров

GET /products/search?q=

Авторизация

POST /auth/login

## Установка и запуск

1. Клонировать репозиторий

git clone https://github.com/NickPodguzov993/adminPanelTest.git

2. Перейти в папку проекта

cd adminPanelTest

3. Установить зависимости

npm install

4. Запустить проект

npm run dev

Приложение будет доступно по адресу:

http://localhost:5173
